# 📚 ÖSYM Kaynak Öneri Web Sitesi

Bu proje, TYT, AYT, KPSS gibi sınavlara hazırlanan öğrenciler için kaynak önerileri sunan bir web sitesidir.

## 🚀 Özellikler
- TYT, AYT, KPSS gibi sınavlara göre kategori sistemi
- Kitap önerileri ve görseller
- Basit ve kullanıcı dostu arayüz
- Sayfa bazlı içerik yapısı

## 🛠️ Kullanılan Teknolojiler
- HTML
- CSS

## 📌 Açıklama
Bu proje ile web tasarım becerileri geliştirilmiş ve kullanıcıların sınavlara yönelik kaynaklara kolay erişmesi amaçlanmıştır.

## 📷 İçerik
Projede farklı sınavlara ait kitap önerileri ve görseller kullanılmıştır.
